주식 종합 분석 보고서를 생성합니다.

분석 대상 티커: $ARGUMENTS

## 실행 절차

1. 먼저 의존성이 설치되어 있는지 확인하고, 없으면 설치합니다:
   ```
   pip install -r requirements.txt
   ```

2. 통합 CLI를 통해 각 분석을 병렬로 실행합니다:
   ```
   python3 src/tools/cli.py fundamental $ARGUMENTS
   python3 src/tools/cli.py technical $ARGUMENTS
   python3 src/tools/cli.py news $ARGUMENTS
   python3 src/tools/cli.py risk $ARGUMENTS
   python3 src/tools/cli.py earnings $ARGUMENTS
   ```

3. JSON 결과를 파싱하여 아래 형식의 **한국어 종합 분석 보고서**를 작성합니다.

## 보고서 형식

### 📊 [회사명] ($ARGUMENTS) 종합 분석 보고서

#### 1. 기업 개요
- 회사명, 섹터, 산업, 시가총액, 현재 주가 등 기본 정보를 정리

#### 2. 재무 분석
- **수익성**: 매출, 영업이익, 순이익, 영업이익률, 순이익률, ROE, ROA
- **성장성**: 매출 성장률, 이익 성장률 (YoY 기준)
- **재무 건전성**: 부채비율, 유동비율, 이자보상배율
- **현금흐름**: 영업CF, 투자CF, 재무CF, FCF
- **밸류에이션**: P/E, Forward P/E, P/B, P/S, EV/EBITDA, PEG, 배당수익률
- 각 항목에 대해 수치와 함께 이모지로 판단을 표시:
  - ✅ 양호 (긍정적 수치)
  - ⚠️ 주의 (경계 수준)
  - 🔴 우려 (부정적 수치)

#### 3. 기술적 분석
- **RSI**: 현재 값과 과매수/과매도 판단
- **MACD**: MACD선, 시그널선, 히스토그램 상태 및 크로스오버
- **볼린저 밴드**: 현재가 위치 (%B), 밴드폭
- **이동평균선**: SMA(20/50/200) 대비 현재가 위치, 골든/데드크로스
- 각 지표별 시그널을 이모지로 표시:
  - 📗 매수
  - 📕 매도
  - ➖ 중립

#### 4. 실적발표 일정 및 컨센서스
- **다음 실적발표일**: 날짜 (D-day 표시)
- **컨센서스 EPS**: Average / High / Low
- **컨센서스 매출**: Average / High / Low
- **배당 일정**: 배당금, 배당락일, 지급일
- 실적발표 임박(2주 이내) 시 ⚠️ 경고 표시

#### 5. 리스크 분석
- **변동성**: 일간/연간 변동성
- **VaR**: 95%, 99% 일일 Value at Risk
- **최대 낙폭**: 분석 기간 내 Maximum Drawdown
- **베타**: 시장(SPY) 대비 변동성
- **샤프 비율**: 위험 대비 수익
- **리스크 등급**: 높음/중간/낮음

#### 6. 뉴스 분석
JSON의 `news.news_items` 데이터를 기반으로 다음을 작성합니다:

- 각 기사의 **제목 + 본문**을 직접 읽고 내용을 분석
- 각 기사별 판정을 이모지와 함께 표시: ✅ 긍정 / 🔴 부정 / ➖ 중립 (한 줄 요약과 함께)
- **주요 긍정 요소**: 뉴스에서 발견된 호재 요약 (구체적 기사 근거 포함)
- **주요 부정 요소**: 뉴스에서 발견된 악재 요약 (구체적 기사 근거 포함)
- **시장 심리 평가**: 뉴스 전반에서 읽히는 시장 분위기 종합 판단
- 뉴스가 없는 경우 "분석 기간 내 관련 뉴스 없음"으로 표시

#### 7. 종합 투자 의견
- **강점**: 재무/기술적/뉴스 분석에서 도출되는 긍정적 요소
- **약점**: 재무/기술적/뉴스 분석에서 도출되는 부정적 요소
- **리스크 요인**: 주의해야 할 위험 요소 (뉴스에서 발견된 리스크 포함)
- **종합 판단**: 현재 시점에서의 투자 매력도 평가 (뉴스 심리 반영)

#### 8. 투자 결정: Yes or No

모든 분석을 종합하여 **현 시점 투자 여부를 Yes 또는 No로 명확하게 제시**합니다.

- **결정**: ✅ Yes 또는 ❌ No 로 표시
- **핵심 근거**: 결정을 뒷받침하는 가장 중요한 2~3가지 이유를 간결하게 제시
- **핵심 수치 테이블**: 결정에 가장 큰 영향을 준 재무/기술적 지표를 테이블로 정리 (지표, 수치, 의미)
- **진입 조건** (No인 경우): 향후 어떤 조건이 충족되면 재검토할 수 있는지 구체적으로 명시
- **주의 사항** (Yes인 경우): 투자 시 반드시 모니터링해야 할 리스크와 손절 기준 제시

## 주의사항
- 모든 내용은 한국어로 작성
- 큰 숫자는 읽기 쉽게 표현 (예: 3조 2,000억원, $3.2T)
- 데이터가 없는 항목은 "N/A"로 표시하고, 가능한 경우 이유를 설명
- 이 보고서는 투자 참고 자료이며 투자 권유가 아님을 명시
