투자 용어 한글화 규칙 — 보고서·메모의 한국어 가독성 표준. (`/invest:glossary`로 조회, 다른 분석 스킬이 보고서 작성 시 참조)

> **목적**: 영어를 직역·음차·미번역한 표현이 한국어 보고서에 섞여 한국 사용자가 의미를 알기 어려운 문제를 막는다. 보고서·요약(summary.md)·종합보고서(report.md)에 공통 적용한다.

---

## 핵심 규칙

1. **영어 단어를 한글 문장에 그대로 박지 말 것.** `net liquidity`·`conviction`·`payoff`·`re-rating`·`dry powder` 같은 단어는 한글 표준어로 옮긴다. 부득이하면 **첫 등장 시 "한글(원어)"로 한 번만 병기**한 뒤 한글을 쓴다.
2. **불투명한 직역·신조어 금지.** 영어 관용구를 글자 그대로 옮겨 한국인에게 뜻이 안 통하는 표현을 만들지 않는다(예: hard money → "디지털 경화" ✗ → "디지털 금/가치저장 자산" ○, strong/weak hands → "강한 손/약한 손" ✗ → "장기 보유세력/단타세력" ○).
3. **음차(transliteration)는 표준어가 있으면 표준어 우선.** 리레이팅 → 멀티플 재평가, 셋업 → 여건/구도, 캐스케이드 → 연쇄.
4. **철자·표기 통일.** 같은 보고서 안에서 한 단어를 두 가지로 적지 않는다(다이버전스 ✓ / 디버전스 ✗).
5. **지표 약어는 첫 등장 시 풀이 후 약어 사용.** NVT·MVRV·DCF·VaR·MDD·RS 등은 처음에 한 번 뜻을 적고(예: "MVRV(시가총액÷실현가)"), 이후 약어로 쓴다.

## 대응표

### A. 미번역 영어 → 한글
| 쓰지 말 것 | 쓸 것 |
|---|---|
| net liquidity | 순유동성 |
| dry powder | 대기 유동성(실탄) |
| conviction | 확신도 |
| payoff / payoff ratio | 손익비 |
| re-rating / de-rating | 멀티플 재평가(상향/하향) |
| pro-cyclical | 경기순응적 |
| breakout / accumulate | 돌파(매수) / 분할매수 |
| capitulation | 투매·항복 |
| reserve(기업) | 보유고/적립 |
| surge / flat | 급증(배수) / 보합·횡보 |
| falsifier | 반증 조건 |
| breadth | (자금흐름) 폭 |
| moat | 해자(진입장벽) |

### B. 불투명 직역 → 한글
| 쓰지 말 것 | 쓸 것 |
|---|---|
| 디지털 경화 (hard money) | 디지털 금 / 가치저장 자산 |
| 강한 손 / 약한 손 | 장기 보유세력 / 단타세력 |
| 무거운 천장 (heavy ceiling) | 상단 저항(천장) |
| 희석 오버행 | 희석 대기물량 |
| 반사성 (reflexivity) | 가격-펀더멘털 되먹임(반사성) — 첫 등장 시 풀이 |

### C. 음차 → 표준어
| 쓰지 말 것 | 쓸 것 |
|---|---|
| 리레이팅 | 멀티플 재평가 |
| 셋업 | 여건/구도 |
| 캐스케이드 | 연쇄(청산) |
| 디레버리징 | 레버리지 축소 |
| (섹터) 로테이션 | 순환매 |
| 하드스톱 | 고정 손절 |
| 트랜치 | 분할분 |
| 컨글로머릿 디스카운트 | 사업복합 할인 |
| 풀사이즈 / 클라이맥스 | 전량 / 정점 |
| 바이어스 (롱 바이어스) | 쏠림 (매수 쏠림) |
| 디스카운트 | 할인 |
| 디버전스 | 다이버전스(괴리) |

### 유지 허용 (한국 시장 정착어)
모멘텀·레짐·ETF·RSI·MACD·볼린저·피라미딩·트레일링스톱, 그리고 NVT·MVRV·TVL 등 크립토 지표(첫 등장 풀이 조건). 4분면 레짐 라벨은 한글 우선(리플레이션·골디락스·스태그플레이션·디스인플레이션·과열).
