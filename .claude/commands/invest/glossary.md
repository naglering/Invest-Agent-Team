투자 용어 한글화 규칙 — 보고서·메모의 한국어 가독성 표준. (`/invest:glossary`로 조회, 다른 분석 스킬이 보고서 작성 시 참조)

> **목적**: 영어를 직역·음차·미번역한 표현이 한국어 보고서에 섞여 한국 사용자가 의미를 알기 어려운 문제를 막는다. 보고서·요약(summary.md)·종합보고서(report.md)에 공통 적용한다.

---

## 핵심 규칙

1. **영어 단어를 한글 문장에 그대로 박지 말 것.** `net liquidity`·`conviction`·`payoff`·`re-rating`·`dry powder` 같은 단어는 한글 표준어로 옮긴다. 부득이하면 **첫 등장 시 "한글(원어)"로 한 번만 병기**한 뒤 한글을 쓴다.
2. **불투명한 직역·신조어 금지.** 영어 관용구를 글자 그대로 옮겨 한국인에게 뜻이 안 통하는 표현을 만들지 않는다(예: hard money → "디지털 경화" ✗ → "디지털 금/가치저장 자산" ○, strong/weak hands → "강한 손/약한 손" ✗ → "장기 보유세력/단타세력" ○).
3. **음차(transliteration)는 표준어가 있으면 표준어 우선.** 리레이팅 → 멀티플 재평가, 셋업 → 여건/구도, 캐스케이드 → 연쇄.
4. **철자·표기 통일.** 같은 보고서 안에서 한 단어를 두 가지로 적지 않는다(다이버전스 ✓ / 디버전스 ✗).
5. **지표 약어는 첫 등장 시 풀이 후 약어 사용.** NVT·MVRV·DCF·VaR·MDD·RS 등은 처음에 한 번 뜻을 적고(예: "MVRV(시가총액÷실현가)"), 이후 약어로 쓴다.

## 문체 규칙 (한국어 보고서 작성)

> **목적**: 어휘를 한글화해도 문장이 "비(非)증권가스러우면" 리포트로 안 읽힌다. 아래는 실제 한국 증권사 리서치(미래에셋·KB 등) 문체 관습을 옮긴 작성 표준이다. 어휘 규칙과 함께 보고서·summary.md·report.md에 공통 적용한다.

1. **본문은 완결 평서체(`~다`)로 쓴다.** 산문 문단은 "~할 전망이다 / ~로 판단한다 / ~을 추천한다 / ~으로 추정된다"로 끝낸다. 개조식(`~음`/`~함`)은 **표 라벨·헤더 박스에만** 짧게 쓴다(본문 산문에 개조식 남발 금지).
2. **두괄식이 원칙.** 각 섹션 첫 문장에 결론을 먼저 박고 근거는 뒤에 댄다. Executive Summary 첫 줄은 **투자의견 + 목표주가 + 핵심 숫자**로 시작한다("배경부터 길게" 미괄식 금지).
3. **모든 판단에 근거를 한 문장 안에 붙인다.** "과도하게 반영됐다고 판단한다"처럼 주장과 근거 프레임을 함께 쓴다. 근거 없는 단정("좋다", "오른다")은 금지.
4. **숫자엔 증감·비교 기준을 항상 병기한다.** 단독 수치를 쓰지 않는다 — `영업이익 830억원(−25% YoY)`, `매출 6,940억원(+11% QoQ)`처럼 절대값 + 괄호 안 증감률이 한 세트. 추정치엔 `25F`/`26F`(F=전망), 직전 12개월엔 `LTM`을 붙인다. 컨센서스 대비는 `상회/부합/하회`로 적는다.
5. **불확실성은 헤지어로 단계 조절한다.** 강도 순: `~할 전망이다` ≈ `~로 판단한다` < `~가능성이 높다` < `~할 수 있는 구간이다` < `~으로 보인다`(약). 확정형(`~할 것이다`)은 확신이 높을 때만 제한적으로 쓴다. 리스크는 본문 낙관 톤에 섞지 말고 조건부("~이어지지 않는다면 ~ 가능성")로 균형 잡는다.
6. **투자의견·목표주가(TP)는 정형 표기.** 변동 방향을 괄호로 명시한다 — `투자의견(신규/유지/상향/하향) 매수`, `목표주가(상향) ▲$430`, `현재가 $312`, `상승여력 +37.8%`. 그냥 "매수"로 끝내지 않는다.
7. **목표주가 산정은 공식 한 문장으로.** "적정주가 X원"으로 던지지 말고 **`기준연도 실적 × 타겟 멀티플(적용 이유)`** 로 적는다 — 예: "26F EPS에 타겟 P/E 22배(피어 평균)를 적용해 목표주가 $430을 제시한다". 음수 FCF·자산주는 PBR(BPS × 호황기 배수) 등 대체 근거를 명시한다.
8. **권장 증권가 어휘를 쓴다**(아래 표). 밋밋한 일반 동사 대신 정착 표현으로 톤을 살린다.
9. **영어는 정착 약어에만 한정.** TP·YoY·QoQ·P/E·P/B·EV/EBITDA·OPM·Top Pick·F·LTM 정도만 허용하고, 서술은 한글로 쓴다(어휘 한글화 규칙과 동일). 문장에 영어 형용사를 끼워 넣지 않는다.
10. **표·그림 캡션은 명사구로.** `표 N. NAVER 분기 실적 전망`, `그림 N. 피어 밸류에이션 현황` 형식(번호 + 명사구, 서술문 금지). 표 우상단에 단위 `(십억원, 배, %)`, 하단에 `자료: ○○` 출처를 단다. (PDF의 Exhibit 자동 번호와 별개로 markdown 본문에도 적용.)
11. **카탈리스트는 날짜·고유명사로 적시한다.** "향후 호재 기대" ✗ → "5/15 출시한 세븐나이츠 리버스", "3분기 ETF 승인 심사" ○.
12. **이모지는 산문에서 빼고 표·박스에만.** 산문 문단은 평서체 문장으로 쓰고 🟢🟡🔴·섹션 아이콘을 박지 않는다. 판정 표의 신호 칸·판정 박스·게이지에는 기존 이모지 팔레트를 유지한다.

### D. 권장 증권가 어휘 (밋밋한 표현 → 증권가 표현)
| 밋밋한 표현 | 증권가 표현 |
|---|---|
| 주가가 다시 오를 것 | 멀티플 재평가(리레이팅) 본격화 전망 |
| 실적이 좋아짐 | 실적 모멘텀 |
| (주가가) 싸다 | 밸류에이션 매력(밸류 매력) |
| 예상보다 좋은 실적 | 어닝 서프라이즈 / 컨센서스 상회 |
| 예상대로 / 기대 이하 | 컨센서스 부합 / 컨센서스 하회 |
| 오를 여지 | 상승여력(업사이드) |
| 가장 선호하는 종목 / 그다음 | 최선호주(Top Pick) / 차선호주 |
| 비중 늘려라 / 줄여라 | 비중확대 / 비중축소 |
| 이득 보는 종목 | 수혜주 |

### 비증권가 실수 체크리스트 (보고서 제출 전 자가점검)
- [ ] 본문을 개조식(~음/~함)으로 쓰지 않았다 — 평서체(~다)인가
- [ ] 결론을 섹션 첫 문장에 뒀다 — 미괄식이 아닌가
- [ ] 모든 핵심 숫자에 `(±% YoY/QoQ)`·`25F`·`상회/하회` 등 기준이 붙었나
- [ ] 투자의견에 변동 방향(신규/유지/상향/하향)과 상승여력을 명시했나
- [ ] 목표주가에 `기준실적 × 타겟멀티플(이유)` 산정 한 문장이 있나
- [ ] 단정 대신 헤지어로 강도를 조절했나(특히 리스크)
- [ ] 산문 문단에 이모지·불필요한 영어를 박지 않았나
- [ ] 표 캡션이 `표 N. 명사구` + 단위 + 자료 출처를 갖췄나
- [ ] 카탈리스트를 날짜·고유명사로 적시했나
